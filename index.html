What is Svetch?

Svetch is a front end proto-typing tool used to create full SvelteKit applications! Create your project skeleton and export locally or to github!

Create the SvelteKit project framework of your dreams in minutes!

About Svetch

Create Svelte Components and add them to the canvas!

Add your components, move them, resize them, nest them, and delete them. Don't worry, we are tracking their heirarchy!

Click on a component to see the code generated in it's Svelte component file!

Whether you click on the canvas or a component, we will provide to you what code would be inside its created file inside the code block!

Visualize your parent-child hiearchy using our Tree graph

As you create, your components relationships will be tracked with every move! Toggle the canvas to see your component hierarchy using our D3 tree!

Export your SveleKit project skeleton! It is ready to run after an NPM install!

Your project can be exported locally or to a Github repo using Github OAuth. This project will contain every file you need to run your app. In other words, you now have a SvelteKit project! Now it is time for yout oadd your content and styling in your own editor. No need to create new files!
